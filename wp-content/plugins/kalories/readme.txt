Kalories application

== Description ==

Kalories application is a website where you can see a list of your meals and calories. Calories are entered manually.
Each entry could be edited and deleted.

An entry has the following fields:
- date
- time
- text
- number of calories

User can filter date (from-to).
For example: how much calories have I had in the last week?

User can set the expected number of calories per day in a settings' panel.
When displayed, the total for that day is colored in green, otherwise it is red.

= Usage =

1. Login as user
2. Set your daily calories threshold
3. Add, modify or delete your meals
3. Use the filter to evaluate how much calories you have eaten in a period

Links: [Author's Site](http://www.andrealandonio.it)

== Installation ==

1. Unzip the downloaded `kalories` zip file
2. Upload the `kalories` folder and its contents into the `wp-content/plugins/` directory of your WordPress installation
3. Activate `kalories` from Plugins page
